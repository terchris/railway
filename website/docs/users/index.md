# Brukerguider

Velkommen. Railway er Oslo Røde Kors' verktøy for frivilligregistrering. Disse guidene viser hvordan du bruker systemet — én side per type bruker.

## Hvem er du?

### Frivillig eller ny bidragsyter

Du vil melde deg som frivillig eller medlem og lurer på hvordan registreringsskjemaet fungerer.

→ [Slik melder du deg på](public-registration.md)

![Det åpne registreringsskjemaet på `/`](/img/screenshots/rwg-pub-home.png)

### Stab / administrasjon

Du logger inn på `/admin` for å se påmeldte, redigere aktiviteter, evalueringsspørsmål, medlemskap eller systeminnstillinger. Det finnes flere stab-roller med ulike tilganger — fra full administrator til snevrere roller som «kun registreringer» eller «kun innhold».

→ [Administrasjonsguider](admin/index.md)

![Innloggingssiden viser en rollevelger som beskriver hva hver rolle kan](/img/screenshots/rwg-adm-login.png)

## Hvor finner du hjelp ellers?

- **Spørsmål om innholdet på skjemaet** (aktiviteter, evalueringsspørsmål, medlemskap): kontakt din lokale Røde Kors-administrator.
- **Tekniske problemer** (siden laster ikke, feilmelding på sending, innlogging fungerer ikke): be administratoren videresende til ansvarlig utvikler. Lenke til feilsøkingsdokumentasjon ligger nederst på hver guide.
- **Utviklere som setter opp eller endrer Railway**: se [Bidragsytere](../contributors/).
