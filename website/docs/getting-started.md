# Velkommen til Railway

Railway er Oslo Røde Kors' verktøy for **frivilligregistrering** — et åpent påmeldingsskjema for nye frivillige og medlemmer, og et administrasjonsgrensesnitt for staben som tar imot påmeldingene.

## Om denne nettsiden

Brukerguidene under [Brukerguider](users/) er på norsk og er skrevet for sluttbrukere. Utviklerdokumentasjonen ([Bidragsytere](contributors/) og [AI-utvikler](ai-developer/)) er på engelsk og dekker hvordan koden er bygget opp, hvordan databasen er strukturert, og hvordan du kan bidra med endringer.

**Stab-roller:** Administrasjonsdelen er delt i fem stab-roller med ulike tilganger — fra full administrator til snevrere roller som «kun registreringer» eller «kun innhold». Se [Administrasjon](users/admin/index.md) for valgvelgeren og hvilken rolle som passer din jobb.

**Spørsmål om innholdet** (aktiviteter, evalueringsspørsmål, medlemskap): kontakt din lokale Røde Kors-administrator.

**Tekniske problemer** (siden laster ikke, feilmelding på sending, innlogging fungerer ikke): kontakt administrasjonen og be om at saken eskaleres til ansvarlig utvikler.
