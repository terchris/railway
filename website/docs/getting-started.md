# Velkommen til Railway

Railway er Oslo Røde Kors' verktøy for **frivilligregistrering** — et åpent påmeldingsskjema for nye frivillige og medlemmer, og et administrasjonsgrensesnitt for staben som tar imot påmeldingene.

## Hvem er du?

| Du er | Start her |
|---|---|
| **Frivillig eller medlem** som vil melde deg på | [Slik melder du deg på](users/public-registration.md) |
| **Stab / administrasjon** som tar imot påmeldinger | [Administrasjonsguider](users/admin/index.md) |
| **Utvikler** som setter opp eller endrer Railway | [Bidragsytere](contributors/index.md) |

## Om denne dokumentasjonen

Brukerguidene under [Brukerguider](users/) er på norsk og er skrevet for sluttbrukere. Utviklerdokumentasjonen er på engelsk og dekker hvordan koden er bygget opp, hvordan databasen er strukturert, og hvordan du kan bidra med endringer.

Dokumentasjonen er under utbygging. Se [planen](ai-developer/plans/active/PLAN-user-documentation.md) for status.
