# Railway

Verktøy for **frivilligregistrering** for Oslo Røde Kors. Sluttbrukerne er volontører som melder seg som frivillige eller medlemmer, og staben som tar imot påmeldingene.

## Kom i gang

| Du er | Start her |
|---|---|
| **Frivillig eller medlem** som vil melde deg på | [Slik melder du deg på](users/public-registration.md) |
| **Stab / administrasjon** som tar imot påmeldinger | [Administrasjon](users/admin/index.md) |
| **Utvikler** som setter opp eller endrer Railway | [Bidragsytere](contributors/index.md) |

## Om denne dokumentasjonen

[Brukerguider](users/) er på norsk og er skrevet for sluttbrukere — frivillige som melder seg, og staben som tar imot. [Bidragsytere](contributors/) og [AI-utvikler](ai-developer/) er på engelsk og dekker hvordan koden er bygget opp, hvordan databasen er strukturert, og hvordan du kan bidra med endringer.

Se [Velkommen-siden](getting-started.md) for en lengre introduksjon med video.
